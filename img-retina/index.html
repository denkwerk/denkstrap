Retina image

Cross browser implementation of retina images for high density screens (smartphones, tablets, Retina MacBook Pro, 4K displays)

<img src="{normal image}" srcset="{retina image} 2x" width="{normal width} height="{normal height}">

Based on W3C standard srcset, already supported by Chrome.
http://www.webkit.org/demos/srcset/

JavaScript polyfill: jquery-srcset-retina-polyfill
https://github.com/jcampbell1/jquery-srcset-retina-polyfill

Example

On a high density screen the image shows [retina dpi], otherwise [normal dpi]
