Basic carousel with default settings

Default settings:

- auto slideshow: disabled
- speed: 1000 (speed of prev and next transitions in milliseconds)
- continuous: enabled/true (create an infinite feel with no endpoints)
- pagination: disabled
- slide-counter: disabled (pagination, e.g. "2/5")
- next-button-class: ".carousel-next" (navigation button)
- prev-button-class: ".carousel-prev" (navigation button)

Custom settings:

These settings can be overriden by data attributes added to the carousel element (element with class="carousel"):

- data-auto-duration="8000" - auto slideshow (time in milliseconds between slides)
- data-slide-speed="1000" - slide/transition speed
- data-continuous="false" - disable infinite sliding
- data-show-pagination="true" - pagination
- data-show-slidecounter="true"
- data-next-class=".myclass" (custom next-button selector)
- data-prev-class=".myclass" (custom prev-button selector)
- custom start slide: item with class="start-item" becomes first item in carousel

Events:

'carouselChange' -> params: (event, currentItem)
demo usage:
$('.carousel').on('carouselChange', function(e, currentItem){})
